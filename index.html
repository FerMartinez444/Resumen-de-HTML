Etiquetas para imágenes

Las etiquetas para imágenes se les llama con img, donde dentro de ella tendremos el src y el alt (entre otros)
el src sirve para poner el link donde se encuentra la imagen a la que queremos mostrar y el alt sirve para ponerle un nombre a la imagen.
El alt es por las dudas de que no nos cargue la imagen al entrar a la pagina, nos aparecería un texto indicando qué contenido había.
Dentro de la etiqueta img, también tenemos el with y height que nos sirve para ajustar el tamaño de la imagen, por si esta es muy grande o muy chica.

Etiquetas para videos

Misma semántica que para las imágenes, solamente que la etiqueta en este caso es video, dentro de esta etiqueta podemos poner muchas cosas,
como el muted (para que el video no se escuche), autoplay (para que cuando entremos a la página se reproduzca solo el video, controls (sirve para que nosotros podamos controlar el video
dándole pausa, subiendo y bajando el volumen y desplazándonos por el mismo). En este caso, el video tiene la palabra controls para que podamos controlarlo con mayor facilidad.

Tablas en HTML

Para hacer tablas en html, tenemos que saber que se separa en bloques, donde el inicio y el fin de cada bloque se le da con la etiqueta table
Dentro del bloque table tenemos 3 bloques de menor jerarquía llamados thead, tbody, tfoot, donde el thead son los elementos cabeceras de la tabla.
Dentro de cada bloque, tenemos la etiqueta tr la cual genera una fila nueva cada vez que lo abrimos. Dentro de ella, y dependiendo del bloque donde estemos posicionados, tenemos
los th, td donde estos nos ayudan a poner elementos dentro de cada columna en una fila en particular. th se usa para el bloque thead y td para los bloques tbody y tfoot

Formularios en HTML

Para crear un formulario primero tenemos que generar un bloque llamado form donde entraran y se validarán todos los comandos
utilizados para los formularios. Existen muchos comandos para los formularios, los mas importantes son label e input, donde el primero
será para generar el cuadrado que pida la información y el imput sera que tipo de informacion permite ese cuadrado.
Dentro del input, se debe pedir el tipo de información, con la etiqueta type="tipo de información" y también podemos poner un ejemplo dentro del bloque
que es lo que se verá en texto gris indicando un ejemplo de la informacion que aceptará ese bloque input, este comando se llama placeholder
Tambien, dentro de los formularios existen los button, que son botones que realizan acciones, en este caso, el button será de tipo submit.

    Main
